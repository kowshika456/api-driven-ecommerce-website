# Shopping Bazaar - API Driven E-Commerce Website

## Overview

A full-stack e-commerce web application developed using Python, Django, SQLite3, HTML, CSS, and JavaScript. The application allows users to browse products, manage their cart, and perform e-commerce operations.

## Features
- User Registration and Login
- Product Catalog
- Add to Cart
- Order Placement
- Product Search
- Admin Panel
- CRUD Operations

## Technologies Used
- Python
- Django
- SQLite3
- HTML5
- CSS3
- JavaScript
- Bootstrap

## Installation
1. Clone the repository
2. Install dependencies
pip install -r requirements.txt
3. Run migrations
python manage.py migrate
4. Start the server
python manage.py runserver
